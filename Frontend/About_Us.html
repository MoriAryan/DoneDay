About This Website

What is this website?

This is a simple habit tracking web app built to help students and individuals stay consistent with their daily goals and routines.

Why it was made

We all have good intentions, but tracking progress is often messy and gets forgotten. This tool lets you visually track your habits every day, helping you stay motivated, build discipline, and see your growth over time.

How does it work?

  1. Add Habits: You can add any habit you want to track (like studying, drinking water, working out, etc.).
  2. Mark Daily Progress: Each day, check off which habits you completed.
  3. View Weekly Summary: See your entire week’s progress in one table with ✔️ and ❌ for each habit.

Key Features

  • Simple Interface: No login or complicated setup. Just start tracking.
  • Weekly Overview: Get a clear table view of your habit completion across the week.
  • Progress Percentage: See how consistent you’ve been each day with a progress score.
  • Real-Time Updates: All data is saved and reflected instantly in the weekly progress view.

Who is it for?

This website is especially useful for students, self-improvement seekers, or anyone trying to build good daily habits.

Tech Stack Used

  • Frontend: HTML, CSS, JavaScript
  • Backend: Node.js with Express (for APIs)
  • Data Storage: JSON file / Local or future database support
